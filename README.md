# Charles S Portfolio

> A polished personal portfolio built with HTML, CSS, and vanilla JavaScript, featuring animated sections, project modals, skill badges, and 3D-inspired interactive background effects.

## Overview

This repository contains a static portfolio site for Charles S. The site is designed for easy editing, local preview, and static deployment.

## Highlights

- Responsive single-page portfolio layout
- Animated hero intro with typewriter effect
- Project gallery with detail modal overlays
- Interactive skills section with icon badges
- 3D canvas-style background visuals using Three.js
- Smooth animations with GSAP + ScrollTrigger

## Demo

Open locally:

`portfolio/index.html`

For best results, serve the folder over HTTP instead of opening via `file://`.

## Built With

- HTML5
- CSS3
- JavaScript (vanilla)
- Three.js
- GSAP + ScrollTrigger
- Icon fonts / Devicons
